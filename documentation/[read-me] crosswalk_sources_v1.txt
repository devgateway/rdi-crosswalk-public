World Bank Microdata Library
-------------------
Source: http://microdata.worldbank.org/index.php/catalog/central
Public: Yes (some datasets restricted)
Indicators: Almost 1900 different datasets from different sources; health-related collections with counts are included below:
	1. WHO (72): http://microdata.worldbank.org/index.php/catalog/WHO (only World Health Surveys from 2003 for individual countries; WHO link here: http://www.who.int/healthinfo/survey/en/)
	2. DHS (262): http://microdata.worldbank.org/index.php/catalog/dhs (DHS link here: http://dhsprogram.com/data/available-datasets.cfm)
	3. Global Health Data Exchange (GHDx) (21): http://microdata.worldbank.org/index.php/catalog/ghdx (better and more data here: http://ghdx.healthdata.org/)
	4. Other limited options (mostly single country/ single year datasets)
Point of contact: data@worldbank.org
Gathering process: [...]
Description: Limited source of external datasets; better to track them down from the original sources

WHO Global Health Observatory (GHO)
-------------------
Source: http://www.who.int/gho/en/
Public: Yes
Indicators: seriously thousands...
Metadata: http://apps.who.int/gho/data/node.metadata.18?lang=en also, here is a link that explains how to extract raw files from the website http://apps.who.int/gho/data/node.resources.examples?lang=en
Point of contact: gho_info@who.int
Gathering process: [...]
Local file: "META_WHO dimensions.xlsx", "META_WHO GHO subcat.xlsx", "META_WHO GHO sub_subcat.xlsx"
Description: The first file includes names of the "dimensions" of available data categories and filters, the second is a list of variable categories, and the third is a list of variables measured. Other dimensions listed in the "META_WHO dimensions.xlsx" file such as "COUNTRY" allow for further data filtering.
